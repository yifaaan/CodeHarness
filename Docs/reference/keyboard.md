# 键盘快捷键

CodeHarness TUI 的键盘行为由 TUI 层维护。当前公开文档只记录稳定能力；更完整的快捷键面板和可配置 keybinding 属于后续计划。

## 通用操作

| 操作 | 说明 |
| --- | --- |
| 输入文本并回车 | 提交 prompt 或执行斜杠命令 |
| 输入 `/` | 触发 slash command 输入路径 |
| `Esc` / `Ctrl-C` | 用于中断或退出当前交互状态，具体行为取决于 TUI 状态 |

## 模式切换

通过斜杠命令完成：

```text
/plan
/act
/fullauto
/default
```
